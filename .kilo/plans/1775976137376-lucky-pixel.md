# GramSevá - Complete Project Design Plan

## Project Overview
GramSevá is a full-stack agricultural service platform in Bangladesh that allows farmers to:
- Book local services (electricians, plumbers, doctors, tutors)
- Report village problems (roads, water, electricity)
- Check market prices
- Access via voice commands (Bengali)

---

## Current State Assessment

### Existing Structure
- **Tech Stack**: Next.js 14, TypeScript, Prisma, PostgreSQL, Tailwind CSS
- **Auth**: JWT-based with OTP
- **PWA**: Service Worker for offline support
- **Features**: Services, Reports, Market Prices, Bookings, Voice Search

### Current Design System
- **Primary Color**: `#8B5A2B` (Brown - represents earth/agriculture)
- **Secondary Color**: `#4A7C59` (Green - represents agriculture)
- **Accent Color**: `#FFB347` (Orange)
- **Font**: Hind Siliguri (Bengali support)

---

## Proposed Design Philosophy

### Core Principles
1. **Simplicity** - Minimal cognitive load for rural users
2. **Accessibility** - Large touch targets, high contrast, voice-first
3. **Trust** - Professional, clean interfaces
4. **Speed** - Fast load times, offline-first

### Target Users
- Rural farmers with limited digital literacy
- Service providers (electricians, doctors, tutors)
- Village administrators

## 7. Accessibility Enhancements

### 7.1 WCAG 2.1 AA Compliance
- Color contrast minimum 4.5:1
- Focus indicators visible
- ARIA labels on interactive elements
- Keyboard navigation support

### 7.2 Mobile Accessibility
- Touch targets minimum 44x44px
- No hover-only interactions
- Clear tap states

### 7.3 Voice-First Features
- Prominent microphone button
- Visual feedback when listening
- Command hints displayed
- Error recovery suggestions

## Summary

This redesign focuses on:
1. **Modern, trustworthy appearance** with refined earth-tone palette
2. **Mobile-first usability** with large touch targets
3. **Simplified navigation** with clear hierarchies
4. **Accessibility** for users with limited digital literacy
5. **Performance** with skeleton loading and optimized assets
6. **Voice-first** as key differentiator

The implementation should maintain the current feature set while dramatically improving the user experience for rural Bangladeshi users.
